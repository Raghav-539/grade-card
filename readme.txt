
Welcome to Grade Card Project.

>> This is simple Node Express Project which take user details and marks and then calculate user's average marks and display his/her average marks and grade. 

>> Data entered by user is also being saved in db.json file.

>> Download some modules that are 

    i. node modules - npm i
    ii. npm i nodemon, express, body-parser

>> to run this project simply type this command in your terminal and open https://locahost:4000
    
    i. npm run watch
